************************************************** 详细说明 *************************************************
一、HAL库配置时打开串口，打开DMA RX和TX通道，模式normal，打开串口中断
二、在HAL库的System Core -- NVIC -- Code generation中关闭USART1 global interrupt的 Generate IRQ handler，即不生成串口1的中断处理函数
三、所有串口发送、接收及中断处理、回调函数都在my_dma_usart中，修改接收的最大长度在my_dma_usart.h中修改USART_DMA_RECV_SIZE和USART_RECV_SIZE的宏定义
四、移植仅需根据单片机修改修改my_dma_usart.h中的头文件，在main中引用my_dma_usart.h，并在main函数中对串口进行初始化hal_uart_recv_init();
五、详细配置查看HAL工程
